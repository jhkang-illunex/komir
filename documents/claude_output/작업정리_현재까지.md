## 3. 진단모델 명세 (정확히)

### 3-1. 목적·타깃
- **목적:** 핵심광물 수급위기 조기진단(주간). 과업의 "교사신호 활용" 방침에 따름.
- **타깃(y):** **KOMIS 수급동향지표** (0~100, **낮을수록 위기**). 위기 이진라벨 = 지표 **< 20**.
- ※ 이 지표는 과업이 "유사지표 적합성 비교대상"으로 지정한 것 → 모델은 이를 **재현·개선**하는 구조.

### 3-2. 데이터 기간·표본 (⚠️ AUC의 출처)
- **입력 테이블:** `mart_weekly_diagnosis` → **월간 패널로 집계**(주간 ASOF 자기상관 방지).
- **대상 광종:** CO·CU·LI·NI **4종** (REE는 주간가격 없어 제외).
- **전체 표본:** 2020-01 ~ 2026-06, **월간 312행** (광종4 × 약 78개월).
- **학습:** 2020-01 ~ 2024-12 = **240행** (위기<20: 74건)
- **검증(hold-out):** 2025-01 ~ 2026-06 = **72행** (위기<20: 15건, 위기비율 0.21) — 광종별 각 18개월
- **분할 방식:** **시간순 분할**(무작위 CV 금지 — 지표가 월→주 ffill되어 자기상관 높음).

### 3-3. 사용 컬럼 (피처 9 + 광종)
| 컬럼 | 과업변수 | 비고 |
|---|---|---|
| volatility_12w | ① 시장변동성 | 주간 로그수익률 12주 표준편차(월평균) |
| import_hhi | ② 수입편중도 | 국가별 수입액 HHI |
| import_yoy, import_cagr3 | ③ 수입증가도 | 연간→월 |
| production_hhi | ⑤ 생산독점도 | USGS(커버리지 낮음) |
| spread_pct | 신규 | LME Cash-3M 스프레드(CU·NI만) |
| geopolitical_risk | ⑥ 지정학(광종) | 월간 geo_pressure |
| geo_macro | ⑥ 지정학(거시) | 경제전반 이벤트 |
| ref_price | 가격 | 대표가격 |
| commodity_code | — | 광종(범주) |
> 변수④ 공급부족은 **소비량 미보유로 제외**.

### 3-4. 사용 모델
- **베이스라인:** Naive(광종 평균), Ridge(+광종더미, 표준화·결측대치)
- **주모델:** **HistGradientBoosting** (sklearn) — 회귀 + 이진분류. 결측 native 처리, 광종 범주 반영.

### 3-5. 성능 (검증 = 2025-01~2026-06, 72행)
| 프레이밍 | 모델 | 지표 |
|---|---|---|
| 회귀(지표값) | HistGBM | MAE 14.28 / RMSE 19.43 / R² 0.303 (Naive MAE 17.81 대비 −20%) |
| **위기 이진분류(지표<20)** | HistGBM | **AUC 0.977** |

**➡️ AUC 0.977의 정확한 의미:** 2020~2024 월간 240행으로 학습한 분류기가, **2025-01~2026-06 검증 72행(위기 15건)** 에서 "위기(지표<20) 여부"를 구분한 성능. 학습·검증이 시간으로 분리됨(미래정보 누수 없음).

### 3-6. 한계 (반드시 함께 볼 것)
1. **순환성 주의:** 타깃(KOMIS 수급동향지표)이 본질적으로 **가격 기반**이고 피처에도 가격(ref_price·spread)이 포함 → 높은 AUC는 "가격이 가격기반 지표를 잘 맞힘"을 일부 반영. **완전 독립검증이 아님.** 피처 중요도도 import_hhi·ref_price·spread 지배(지정학≈0).
2. **소표본:** 검증 72행·위기 15건. AUC가 소표본에 민감 → 신뢰구간 넓음. 추가 기간 검증 필요.
3. **월→주 집계:** 지표가 월간이라 주간은 ffill. 실질 해상도는 월간.
4. **지정학 기여 낮음:** 지정학은 지표 *재현*엔 기여 작음. 본 역할(급성 이벤트 조기탐지)은 별도 평가 필요.
5. **REE·변수④ 제외:** REE 주간가격·세계소비량 미보유로 미반영.
