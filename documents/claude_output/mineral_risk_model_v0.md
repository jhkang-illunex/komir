# 광종 수급 리스크 모델 뼈대 v0.1

대상 광종: 구리(cu), 니켈(ni), 코발트(co), 리튬(li), 네오디뮴(nd)
산출물: ① 지정학 이슈 지수(주간) · ② 수급 위기 체계 판단(주간, 4단계) · ③ 국내 수입 추정(월 단위, 12개월 지평)

변수 생산 가능성 범례:
- `[D]` 데이터 직접 (레이크에 있거나 즉시 수집 가능)
- `[C]` 파생 계산 (코드로 생산)
- `[E]` 추정·사전확률 (전문가 anchoring / 사례 캘리브레이션 → 추후 데이터로 교체)

## 0. 공통 구조

### 0.1 상태 변수

| 변수 | 인덱스 | 상태공간 | 생산 |
|---|---|---|---|
| `G[c]` 국가 지정학 상태 | 국가 c (광종 간 공유) | S0 정상 / S1 긴장 / S2 부분차질 / S3 위기 | [C] 필터 산출 |
| `T[c,m]` 무역정책 상태 | 국가 c × 광종 m | T0 정상 / T1 위협 / T2 실질장벽 / T3 봉쇄·금지 | [C] 필터 산출 |
| `F[k]` 공통 인자 상태 | k ∈ {황산·황, 중국미드스트림, 에너지, 해상회랑} | 4단계 동일 | [C] 필터 산출 |
| `W[m,c]` 노출행렬 | 광종 × 국가 생산 점유율 | 연속 [0,1] | [D] USGS/Comtrade, 연 1회 갱신 |

국가 노드 pruning 규칙: 해당 광종 세계 공급 점유율 ≥ 5%인 국가만 노드화.

광종별 특수 배선:
- **코발트**: host-metal 커플링 — `G[DRC]`×구리 조업상태, `G[인니]`×니켈 조업상태를 코발트 모델의 외생 입력으로 추가.
- **네오디뮴**: 황산은 중동 채널이 아닌 `F[중국미드스트림]`에 배선 (중국 내수 황산으로 커버, 노출은 중국 정책 리스크).
- **황산 순환 커플링**: `F[황산·황]`은 외생 + 구리·니켈 제련 상태에서 들어오는 엣지 1개 (제련 부산물 황산).

### 0.2 이벤트 스키마 (텍스트 파이프라인 출력 → 관측 레이어)

```
event: {
  event_id,            # 중복 제거된 고유 사건
  date,
  country,             # ISO code
  mineral,             # cu/ni/co/li/nd/all
  dimension,           # ops(조업) / trade(무역정책) / corridor(회랑) / input(투입재)
  severity,            # 1~4 (LLM 순서형 분류)
  article_count,       # 사건별 기사 수 (주목도 프록시)
  source_ids
}
```

주간 집계 테이블 (모든 하위 모델의 공유 피드):

| 변수 | 산식 | 생산 |
|---|---|---|
| `evt_n[c,m,d,w]` | 주간 unique event 수 | [C] |
| `evt_sev[c,m,d,w]` | 주간 severity 가중합 Σ(severity²·event) — 제곱 가중으로 꼬리 강조 | [C] |
| `art_n[c,m,d,w]` | 주간 기사 수 (salience) | [C] |
| `media_base[c]` | 국가별 평시 기사량 (12개월 이동평균) — 커버리지 편향 정규화용 | [C] |
| `evt_z[c,m,d,w]` | (evt_sev − 평시평균) / 평시표준편차, media_base 정규화 후 | [C] |

## 1. ① 지정학 이슈 지수 (주간, 광종별)

### 1.1 입력 변수

| 변수 | 정의 | 소스 | 생산 |
|---|---|---|---|
| `evt_z[c,m,d,w]` | 0.2 이벤트 z-score | 자체 파이프라인 | [C] |
| `gpr[c]` | 국가별 GPR 지수 (벤치마크·보조 공변량) | Caldara-Iacoviello 공개 데이터 | [D] |
| `oil_brent` | 브렌트 주간 평균 | 시장 데이터 | [D] |
| `freight_war` | 운임·전쟁보험 프록시 (컨테이너 운임지수 등) | 시장 데이터 | [D] |
| `sulfur_spot` | 황/황산 스팟가 | Asian Metal 등 | [D] 수집 |
| `P(G[c]=s)`, `P(T[c,m]=s)`, `P(F[k]=s)` | 상태 확률 (필터) | 모델 내부 | [C] |
| `loss[m,c,s]` | 상태 s에서의 기대 차질 톤수 | 사례 앵커 (Grasberg −35%, Kamoa −8만t, 칠레·페루 파업 −15만t, DRC 황산 −12.5만t 등) | [E] → 사건 축적 시 교체 |
| `annual_supply[m]` | 광종 연간 세계 공급량 | USGS/ICSG | [D] |

### 1.2 조합 방식

**v0 (즉시 코드화, 필터 없이):** 규칙 기반 상태 점수
```
score[c,w]   = EWMA(evt_z[c,·,ops,w], halflife=4주)          # 국가 조업 축
score[c,m,w] = EWMA(evt_z[c,m,trade,w], halflife=4주)        # 무역 축
상태 근사:  P(≥S1)=sigmoid(a₀+a₁·score) 형태의 로지스틱 매핑   # a₀,a₁는 [E] 초기값, 과거 에피소드 2~3개로 눈금 맞춤
```

**v1 (승격):** HMM forward filter — 관측 `evt_z` 벡터, 전이강도 공변량 `oil_brent, gpr, sulfur_spot, freight_war`. 시변 Q는 주 단위 piecewise-constant로 `P_week = expm(Q(x_w)·Δt)`, 주차별 행렬곱 누적.

**지수 산출 (v0/v1 공통):**
```
SaR[m,w]   = Σ_c W[m,c] · Σ_s P(state[c,m]=s) · loss[m,c,s] / annual_supply[m]   # 레벨: supply-at-risk %
tail[m,w]  = 1 − Π_c (1 − P(state[c,m] ≥ S2))^W[m,c]                              # 꼬리: 어딘가 S2+ 확률 (노출 가중)
geo_idx[m,w] = 과거 SaR 분포의 percentile → 0~100 스케일
```
분해(attribution): `contrib[c] = W[m,c]·E[loss]` 를 국가·차원별로 저장 → "누가 몇 포인트" 리포트 자동 생성.

## 2. ② 수급 위기 체계 판단 (주간, 4단계: 관심–주의–경계–심각)

> 요건 반영: 필수 6대 축 체계로 개편. 축 2·3이 **국내(한국) 수입 관점**이므로 ②의 성격은 "글로벌 시장 위기"가 아니라 **국내 수급 안보 위기 지표**로 확정 — ①(글로벌 원인측 위험)과의 역할 구분이 오히려 명확해짐.

### 2.1 필수 6대 축 정의

| # | 축 | 변수 | 산식 | 소스 / 주기 | 생산 |
|---|---|---|---|---|---|
| 1 | 시장 변동성 | `vol[m,w]` | 주간 수익률 26주 실현변동성(연율화). v1: GARCH(1,1) 조건부 변동성 | LME/GFEX/SMM 가격 · 주간 | [C] |
| 2 | 특정국 수입 편중도 | `imp_hhi[m,mo]`, `imp_cr1[m,mo]` | 원산지별 수입 점유율 HHI = Σ s_c², CR1 = 최대 수입국 비중. **12개월 누적 물량 기준** (월별 노이즈 완화) | 관세청 · 월간 | [C] |
| 3 | 국내 수입액 증가도 | `imp_val_yoy[m,mo]` | 수입금액 3개월 이동평균 YoY. 물량 기여 / 단가 기여 분해를 별도 컬럼으로 저장 | 관세청 · 월간 | [C] |
| 4 | 세계 공급부족 | `deficit[m]` | 세계 소비 / 세계 공급 (>1 = 부족) | cu: ICSG 월간 [D] · ni: INSG 월간 [D] · co: Cobalt Institute 연간 [D]+[E] · li·nd: 연간 통계+애널리스트 추정 보간 [E] | [D]/[E] |
| 5 | 생산·매장량 독점도 | `prod_hhi[m,y]`, `resv_hhi[m,y]` | 국가별 세계 생산 점유율 HHI (§0.1 `W[m,c]` 재사용), 매장량 HHI | USGS · 연간 | [C] |
| 6 | 지정학 리스크 | `geo_idx[m,w]`, `tail[m,w]` | ① 산출 그대로 (레벨 + 꼬리) | 모델 내부 · 주간 | [C] |

### 2.2 정규화 및 주기 정합

- **순환 축(1, 3, 4, 6)**: 과거 5년 분위(pct)로 0~1 정규화.
- **구조 축(2, 5)**: 분위 정규화 **금지** — 편중도가 만성적으로 높은 광종(nd 대중국, co 대DRC)에서 분위를 쓰면 위험이 0으로 세탁됨. 절대 임계 매핑 사용, 초기값 [E]:
  `HHI < 0.15 → 0~0.33 / 0.15~0.25 → 0.33~0.66 / > 0.25 → 0.66~1.0 (구간 내 선형)`
- **주기 정합**: 월간(2, 3, 4)·연간(5) 변수는 주간으로 forward-fill 계단함수. **발표 시차 반영한 as-of join 필수** (관세청 익월 중순, ICSG 약 2개월 지연) — 백테스트 look-ahead 방지.
- deficit의 발표 공백기(특히 co·li·nd)에는 재고 커버리지·스프레드를 nowcast 프록시로 대체 가능 [C] (§2.4 보조 패널 재사용).

### 2.3 조합 방식

```
pillar_i[m,w] ∈ [0,1]  (i = 1..6, §2.2 정규화)
crisis_score[m,w] = Σ w_i · pillar_i[m,w]
w 초기값 [E]: 균등 1/6  (발주처 지정 가중이 있으면 그대로 적용, config로 외부화)
등급: 관심 < 0.45 / 주의 0.45~0.60 / 경계 0.60~0.75 / 심각 ≥ 0.75   # [E] 과거 에피소드 눈금 조정
```

- **attribution**: 축별 기여 `w_i·pillar_i` 저장 → 등급 산출 근거 자동 서술.
- **구조/순환 분해 리포트**: 구조축(2, 5) 소계 = 취약도, 순환축(1, 3, 4, 6) 소계 = 스트레스. "구조적으로 취약한 광종에 순환 스트레스 상승 중" 같은 경보 서사가 코드에서 바로 나옴.
- 히스테리시스: 상향 즉시, 하향은 2주 연속 조건 (유지).
- **v1**: 6축을 feature로 ordered logit — 단, 대외 보고 지표는 v0 합성식 유지 (설명가능성·요건 준수), 학습 모델은 내부 검증·가중 보정용.

### 2.4 기존 펀더멘털 변수의 위치

`inv_days`, `spread`, `premium`, `tcrc`는 6축 요건 외 변수이므로 점수 산식에서 제외, **보조 진단 패널**로 이동:
1. 등급 발표 시 근거 설명 (예: "경계 상향 — 백워데이션 심화 동반")
2. §2.2의 deficit nowcast 프록시
3. v1 학습 모델의 추가 feature 후보

### 2.5 라벨 (v1 학습용) — 누수 방지 원칙 유지

v1 ordered logit의 타깃 라벨은 6축 합성식과 독립인 시장 결과 변수로 규칙 생성:
```
label_score[m,w] = 0.4·pct(price_z) + 0.35·(1 − pct(inv_days)) + 0.25·pct(spread)
관심 <0.5 / 주의 0.5~0.7 / 경계 0.7~0.85 / 심각 ≥0.85
```
(합성식에 이미 geo_idx가 들어가므로, 학습 시 자기 산출물로 자기를 맞추는 순환을 여기서 차단.)

## 3. ③ 국내 수입 추정 (월 단위, 12개월 지평, 톤)

### 3.1 타깃 정의

`import_tons[m,month]` = 광종별 HS 품목 바스켓의 월간 수입량(톤). 소스: 관세청 수출입무역통계 [D] (+ UN Comtrade 교차검증).

HS 바스켓 초안 (**코드 반영 전 관세사/HSK 검수 필수** — 기존 HSK RAG로 1차 검증 가능):

| 광종 | HS 후보 |
|---|---|
| cu | 2603(동광·정광), 7402(조동), 7403.11(전기동 캐소드) |
| ni | 2604(니켈광), 7502(니켈 괴), 2833.24(황산니켈) |
| co | 2605(코발트광), 8105(코발트 매트·괴), 2822(코발트 산화물), 황산코발트(2833 세분류 확인) |
| li | 2836.91(탄산리튬), 2825.20(수산화리튬) |
| nd | 2846(희토 화합물), 2805.30(희토 금속), 8505.11(영구자석 — 간접수요 지표로 별도 관리) |

### 3.2 입력 변수

| 변수 | 정의 | 소스 | 생산 |
|---|---|---|---|
| `import_tons[m]` 과거 60개월+ | 타깃 자기시차·계절성 | 관세청 | [D] |
| `battery_ip`, `ev_prod`, `elec_ip` | 이차전지 생산지수, 전기차 생산, 전자부품 IP (다운스트림 활동) | KOSIS·산업활동동향 | [D] |
| `price[m]`, `futures_curve[m]` | 현물가, 선물커브 (선구매/관망 행동) | LME 등 | [D] |
| `fx_usdkrw` | 환율 | 시장 데이터 | [D] |
| `inv_cycle[m]` | 국내 재고 사이클 프록시 = Σ(수입 − 소비추정) 누적의 편차 | 파생 | [C] (소비추정은 [E] 원단위 기반) |
| `geo_path[m]`, `oil_path` | ① MC 경로에서 뽑은 시나리오 경로 | 모델 내부 | [C] |

### 3.3 조합 방식

**v0 (즉시):** 광종별 SARIMAX
```
SARIMAX(import_tons[m]; order/seasonal 자동선택, exog=[battery_ip, price, fx, oil])
→ 12개월 동적 예측 + 신뢰구간
```
**v1 (시나리오 조건부):** ① MC에서 `geo_path`·`oil_path` N개 샘플 → 각 경로를 exog로 조건부 예측 → 경로별 예측 통합 = **팬차트 (P10/P50/P90)**. 지정학 불확실성이 수입 전망 폭에 직접 반영되는 구조.

주의: `geo_idx`가 수입량에 미치는 부호는 광종·국면별로 양가적(공급차질→수입감소 vs 사재기→수입급증). v0에서는 강제로 넣지 말고, v1에서 국면 상호작용항(`geo_idx × price_z`)으로 도입 후 유의성 확인.

### 3.4 동적 가중치 (요건 반영: 위기 상황별 계수 미세조정)

요건의 "어텐션 메커니즘"은 딥러닝 없이 구현 — softmax 가중 평균은 Nadaraya-Watson 커널 회귀와 수학적 동치. 과거 위기 에피소드 표본이 극소(~5–8개)라 학습형 딥 어텐션은 과적합으로 부적합하며, 아래가 정직한 구현 경로:

| 단계 | 방식 | 산식 | 구현 | 생산 |
|---|---|---|---|---|
| v1.5 | 국면 조건부 계수 | `β(t) = β₀ + β₁·crisis_score[m,w]` (또는 ② 등급 더미 상호작용) | SARIMAX exog 상호작용항 (statsmodels) | [C] |
| v2 | 에피소드 어텐션 (커널) | `α_k(t) = softmax(−‖x_t−x_k‖²/τ)`, `β(t) = Σ α_k·β_k` | numpy. τ는 leave-one-episode-out CV | [C]+[E] |
| v3 | 학습형 게이트 | 소형 mixture-of-experts (선형 expert 2~3 + softmax gate) | 에피소드 축적 후에만 — 현 표본으로 보류 | 보류 |

**v2 구성요소:**
- 상황 벡터 `x_t`: ② 6축 정규화값 + 유가 재사용 (Query) — 추가 데이터 불필요
- 에피소드 라이브러리 `{(x_k, β_k)}` (Key, Value): 위기 구간별 국소 회귀 계수. 후보: 2010–11 중국 희토 수출제한(nd), 2022 LME 니켈 스퀴즈(ni), 2022 러시아 침공(ni·기타), 2025 Grasberg(cu), 2026 호르무즈·DRC 쿼터(cu·co). 구간 경계 정의 [E]
- **계수 수축**: 에피소드당 관측이 짧아 β_k가 노이지 → `β_k ← λ·β_k_local + (1−λ)·β₀` (전 기간 계수 β₀로 릿지형 수축, λ는 LOO-CV) — 이상 에피소드 계수로의 어텐션 튐 방지
- attribution: `α_k` 자체가 보고 근거 ("현 상황은 2022 니켈 스퀴즈와 유사도 0.6")
- 검증: leave-one-episode-out (에피소드 단위 홀드아웃)

가격 매핑(밸런스→가격 탄성)에도 동일 구조 적용 — 재고 커버리지가 낮을수록 가격 반응이 볼록해지는 국면 의존성을 v1.5 상호작용 또는 v2 어텐션으로 처리.

| ① 지수 | 실현 차질(톤)과의 선행 상관, 이벤트 스터디 | GPR 국가지수 |
| ② 등급 | RPS, QWK, 혼동행렬, 등급상향 리드타임(일) | 2.2 룰 라벨의 naive 지속 예측 |

### 4.1 학습 진행·적합 진단 산출물 (보고서 그림 매핑)

각 컴포넌트의 최적화 방식이 달라 "학습 곡선"의 형태도 다름. **곡선이 없는 방법론에 epoch 곡선을 인위 생성하지 말 것** — 방법론별 표준 진단으로 대응:

| 컴포넌트 | 최적화 방식 | 학습 곡선 형태 | 보고서 그림 |
|---|---|---|---|
| ② v2 LightGBM | boosting 반복 | 라운드별 train/valid loss (DL 학습곡선과 동형) | loss 곡선 + early stopping 지점 |
| ① v2+ 베이지안 (PyMC) | MCMC / ADVI | MCMC는 loss 없음. **ADVI 실행 시 iteration별 ELBO 곡선** (정당한 학습 곡선) | ELBO 곡선 + trace plot, R-hat, ESS |
| ③ v2 커널 어텐션 | 폐형해 + LOO-CV (반복 학습 없음) | 없음 — 과적합 경로 자체가 부재 | τ(bandwidth) vs CV 오차 U자형 검증 곡선 |
| ② v1 ordered logit · ③ SARIMAX | MLE (Newton/BFGS, 수 회 수렴) | 사실상 없음 | walk-forward 구간별 오차 추이 |

심사 대응 논리: 반복 경사 학습이 없는 컴포넌트는 "폐형해+교차검증 선택 구조로 과적합 경로가 없음"이 학습 곡선보다 강한 방어 논리.

## 5. 구현 순서 제안

1. §0.2 주간 집계 테이블 (텍스트 파이프라인 뒤에 붙는 SQL/pandas 1개)
2. ① v0 EWMA 지수 + attribution — 여기까지 [E]는 `loss` 테이블과 로지스틱 눈금 2개뿐
3. ② v0 6축 합성 지표 (축 2·3은 §3과 관세청 피드 공유 — 수집 잡 1개로 통합, + 보조 패널용 재고·스프레드·프리미엄 수집)
4. ③ v0 SARIMAX
5. ① v1 HMM 필터 승격 → ② v1 ordered logit → ③ v1 시나리오 팬차트
6. v2+: ① 베이지안 계층 사전(PyMC) / ② LightGBM(내부 검증·가중 보정용) / ③ 커널 어텐션 동적 계수

## 6. 최종 구성 (버전 스택)

v0/v1은 폐기가 아니라 v2+의 부품으로 존속:

| 산출물 | 최종 구성 | 비고 |
|---|---|---|
| ① 지수 | v1 HMM·CTMC 필터 코어 + v2+ 베이지안 계층 사전 | v0 EWMA는 필터 이상 감지용 병행 산출 |
| ② 위기 판단 | **대외 보고 = v0 6축 합성식 유지** (요건·설명가능성) | v1/v2는 내부 가중 보정·검증용 |
| ③ 수입·가격 | v1 시나리오 MC 골격 + v2 커널 어텐션 동적 계수 | v0 SARIMAX는 벤치마크로 존속 |

추론 코어는 통계 모델, 딥러닝은 관측 레이어(LLM 이벤트 추출)에 배치 — 위기 에피소드 소표본(~5–8개)과 attribution 요건에 따른 의도적 설계.
