## 0. 프로젝트 개요
- **과업:** AI 기반 핵심광물 수급위기 진단·수요예측 (발주: 광해광업공단/KOMIS). 근거: `260625 ... 과업지시서_일루넥스.pdf`, `260623 ... 착수보고 ver1.pdf`.
- **대상 광종(5):** CU(동)·NI(니켈)·LI(리튬)·CO(코발트)·**REE(희토류, 1종 미정)**.
- **과업 산출물 6개(착수보고 p4 확인):** ①수급위기 진단모델 ②수입 예측모델 ③모니터링 대시보드 ④품목관리·보고서 자동생성 ⑤시계열 AI분석·요약 ⑥생성형AI 대화형검색(챗봇). 모두 KOMIS 연계.
- **착수보고 아키텍처(p6):** 6레이어(입력→Ontology/Graph 구조화·전처리→VSN+LLM+TFT→동적가중 통합→진단·경보 대시보드→정책지원). 챗봇=RAG(Vector DB+LLM Engine)+KOMIS API 연계.

## 3. 리눅스에서 이어가기 (셋업)
```bash
cd mineral_supply_risk
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
python -m scripts.run collect-ecos                    # 한국 거시(검증됨)
python -m scripts.run collect-customs 201301 202512   # 관세청 월간(리눅스=네트워크 OK)
python -m scripts.run features
```
