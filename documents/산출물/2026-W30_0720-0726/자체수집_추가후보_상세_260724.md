# 자체 수집 추가 후보 상세 (2026-07-24, 접근성 실측 포함)

사용자 요청: "자체 수집으로 확보 가능한 데이터를 더 자세히 꼼꼼히 산출". 이 문서의
모든 "실측✓"는 오늘 실제 API 호출로 응답·데이터 존재를 확인한 것이다(불확실 후보를
이론상 가능으로 적지 않음). 우선순위 논리는 이 세션에서 확립된 3원칙:
① 물리량(공급·재고·무역) 우선 — 가격 파생은 오염, ② 히스토리 2016+ — 부분
커버리지는 교란(오늘 거시 백필로 3번째 실증), ③ 광종별 재검정 필수 — 패턴 이식
불가(NI 유의→CU 기각 사례).

현재 격차(후보 선정의 기준): 풀링 보조신호는 한계효용 체감 확인. 남은 가치는
**광종별 축**이다 — CU(전환주 13건, 검정력 최대인데 유의 피처 없음)·REE·CO(재고
부재)·NI(2축 확보, 보강 여지)·LI(데이터 축적기).

## Tier 1 — 즉시 수집 가능(오늘 실측 검증 완료)

### 1. Comtrade 공급국 물리 흐름 4종 ★최우선
기존 `collect_priority_feeds.py`의 Comtrade 경로 재사용(월간, 2016~, 무료).
오늘 실측으로 4종 모두 데이터 존재 확인:

| 흐름 | reporter→partner, HS | 실측 | 광종·의미 |
|---|---|---|---|
| 인도네시아 니켈 수출 | 360→세계, 2604(광석)+7501(매트) | ✓ count=3 (2024-06) | **NI 공급 — 세계 1위 생산국**, 수출금지·쿼터 정책 변수 직결 |
| 호주 리튬광 수출 | 36→세계, 253090(스포듀민)+283691(탄산리튬) | ✓ count=2 | **LI 공급 — 세계 1위**, GFEX 재고와 별개의 상류 축 |
| 칠레 구리정광 수출 | 152→세계, 2603 | ✓ 월 ~25만톤 | **CU 공급 — 세계 1위**(페루 604 추가 확장 가능) |
| 중국←미얀마 희토류 수입 | 156←104, 253090+2846 | ✓ 월 205톤+3,728톤 | **REE 공급리스크 — 미얀마 내전·국경 폐쇄가 중국 중(重)희토류 원료의 최대 리스크 루트**, geo_event 스토리와 정합 |

이로써 5광종 전부에 "최대 공급국의 물리 수출 흐름" 축이 생긴다(기존: 중국 REE
수출·중국←DRC CO 수입뿐). ⚠확인 필요: 국가별 Comtrade 보고 지연(칠레·호주는
2~4개월, 인니는 더 길 수 있음 — 수집 시 최신월 실측해 as-of 지연에 반영).

### 2. CFTC 코발트·리튬 COT(포지셔닝) ★신규 발견
실측✓: CFTC disaggregated(72hh-3qpy)에 **COBALT(188691)·LITHIUM HYDROXIDE(189691)**
존재, 2022-11-22~2026-07-14 주간. CME 상장 코발트금속·수산화리튬 선물의 투기/실수요
포지션 — **CO·LI 최초의 금융 포지셔닝 신호**. 기존 `collect_forecast_exog.py` 확장
1시간 수준. ⚠커버리지 2022-11~(폴드1·2 학습 미커버 — 교란 플래그 필수, 폴드3 중심
검정), CME 계약 유동성이 낮아 OI 절대량 작음(신호 희석 가능성 정직 명시).

### 3. 생산국 통화 환율(공급 유인 메커니즘)
실측: ECB에 IDR(인니 루피아)✓ 존재(2016 값 확인), CLP(칠레 페소)는 ECB 미제공✗.
공급국 통화 약세→달러 표시 수출 유인 증가라는 물리 메커니즘이 있는 비가격 축.
기존 `backfill_macro_history.py`의 ECB 경로 재사용(IDR 즉시, CLP는 칠레 중앙은행
API 별도 실측 필요). 비용 분 단위.

### 4. SHFE·GFEX 선물 거래량·미결제약정(중국판 포지셔닝 대용)
akshare 일별 시세(거래량·OI 포함) — NI 2015~·CU 2005~·LC(리튬) 2023~. COT가 없는
중국 시장의 포지셔닝 프록시. 기존 인프라 재사용, 반나절.

## 종합 의견

Tier 1의 1·2번이 핵심이다 — 기존 수집기 재사용으로 비용이 낮고, 5광종 전부에
"최대 공급국 물리 흐름" 축과 CO·LI 첫 포지셔닝 축을 만든다. 특히 미얀마→중국
희토류 루트는 지정학 이벤트(geo_event)와 물리 흐름을 잇는 유일한 다리다. 다만
기대치는 정직하게: 풀링 동작점 개선보다 **광종별 보조신호·향후 모델 개편 대비
축적**이 주 가치이며, 수집 즉시 채택이 아니라 동일 백테스트 검정을 거친다(원칙).
