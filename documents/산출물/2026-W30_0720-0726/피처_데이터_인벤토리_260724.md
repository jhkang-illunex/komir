# 3모델 피처·데이터 인벤토리 — DB화/미DB화/미수집 후보 (2026-07-24)

사용자 요청: "지수·진단·예측 3모델에 필요한 피처를 ①지금 DB화된 것 ②DB화 안 된 것
③더 있으면 좋은데 수집 안 된 것으로 체크하고, 수집 시 예측/진단력이 향상될 후보를
추릴 것." 근거는 전부 이 세션에서 직접 조회·실측한 것(경로·테이블 명시).

적용 원칙(이 세션의 실증 교훈 2가지):
- **물리량 우선**: 재고·무역흐름·생산 같은 물리 데이터가 전환탐지를 실제로 개선함을
  두 차례 실증(LME 재고 CU·NI: P=1.000 / NI LME+SHFE 2축: P=1.000). 반면 가격 파생
  (원자재지수·가격뉴스 카운트)은 진단 라벨(가격 이격률) 오염으로 두 차례 기각.
- **커버리지 주의**: 히스토리가 짧은 피처(거시 12종, 2021-06+)는 "데이터 존재
  시기=전환 다발기" 교란으로 판정 불가였음 — 2016+ 커버 소스를 우선한다.

## 내 의견(요약)

1. **1~4순위는 전부 수집 권장** — 비용이 작고(1~2번은 기존 수집기 확장 수준),
   이번 세션에서 실증된 "물리량 → 전환탐지 개선" 패턴의 직접 연장이며, 특히 3번
   (Comtrade)은 재고가 구조적으로 없는 REE·CO에 물리 신호를 처음 공급한다.
2. 5~8순위는 예측모델(2-4) 개선 여지가 크지만 설계 비용(as-of 정합성·파싱)이 있어
   1~4 검정 결과를 본 뒤 착수가 합리적.
3. **모든 후보는 수집 즉시 채택이 아니라, 이번에 확립한 동일 백테스트 프레임
   (Δ분류·워크포워드·부트스트랩)으로 검정 후 채택** — E7(가격뉴스)·거시12종처럼
   "좋아 보이는" 피처가 오염·교란으로 기각된 전례가 이미 세 번 있다.
4. KOMIS 파생 지표(시장동향 등)는 라벨 순환 위험이 커서 오염 검정 전 사용 금지.
