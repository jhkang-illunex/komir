# HS코드 → 광종 매핑 검증 리포트

관세청 수출입DB의 광종 분류를 **권위 매핑(`HS코드분류__최종.xlsx` 종합분류 시트)** 과
전수 대조하여 5대 핵심광물 매핑의 정확성·완전성을 검증함.

## 데이터
- 권위 매핑: `HS코드분류__최종.xlsx` > `종합분류` 시트, **HS10 542개** → 광종(Level1) 33종
- 관세청 DB: `(국내)관세청_2013~2025_수출입DB`, **총 105,194행**, distinct (HS,광종명) **571쌍**, 광종명 70종
- 둘 다 **10자리 HS코드** 사용 (형식 일치)

## 검증 결과 (요약)

| 항목 | 결과 |
|---|---|
| 관세청 HS가 권위매핑에 **존재** | 433 / 571 |
| 광종명 **일치** | 427 |
| 광종명 **불일치** | 6 (전부 `창연`=`비스무스`, **동의어 → 실질 0건**) |
| 권위매핑에 **미존재** | 138 (전부 **비핵심 광종**) |
| **5대 핵심광물 미매핑** | **0건 (100% 커버)** ✅ |

## 핵심 결론

1. **5대 핵심광물(동·니켈·리튬·코발트·희토류)은 HS코드 미매핑 0건** — 관세청의 해당 광종 모든 거래행이 권위매핑에 포함됨. 모델 입력에 누락 없음.
2. **실질 충돌 0건** — 유일한 불일치 6건은 `창연`(관세청)과 `비스무스`(매핑)로 **동일 광물의 다른 표기**(창연=Bismuth). 핵심광물 아님.
3. **미매핑 138건은 전부 비핵심**(붕소·고령토·금·바륨·은·석회석·운모 등) → 5종 모델에 영향 없음.
4. 관세청은 희토류를 **단일 광종명 `희토류`**로 관리(원소 분리 안 함). 권위매핑의 희토류 15코드와 정합.

- `hs_commodity_map.csv` — HS10 → 광종(한글)·표준코드·core5여부 (542행, UTF-8-SIG)
- 로더 반영: `dim_hs_commodity` 테이블 + **관세청 `commodity_code`를 HS코드 기준으로 부여**
