## 2. 사유 자동생성 방식

각 경보의 사유 = **공식문안(해당 단계) + 데이터근거 + 지정학 이벤트 인용** 을 규칙으로 조합.
- 지정학 이벤트 유형을 공식 3대 기준에 매핑:
  - 수출통제·제재·전쟁·관세 → **기준① 생산국 정세불안/수출제한**
  - 공급차질(파업·봉쇄·가동중단) → **기준② 시설 재난/수송 차질**
  - 위기지수·가격변동성 → **기준③ 가격 변동성/조달 차질** (항상 적용)
- 해당 광종·월의 최고 severity 지정학 이벤트의 **근거 인용문**을 첨부 → 추적성 확보.

전체 주간 이력의 사유는 `alert_timeline_사유.json`에 저장(광종·주·단계·위기지수·트리거·사유).
