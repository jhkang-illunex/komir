# 수급위기 경보 4단계 산출 로직 리포트

> 근거: 「국가자원안보 확보를 위한 고시」 제7조 · 과업지시서 붙임2
> 단계: **정상 · 관심(Blue) · 주의(Yellow) · 경계(Orange) · 심각(Red)** / 산출주기 주간
> 구현: `claude_output/alert/alert_logic.py` · 결과: `alert_timeline.csv`

---

## 1. 설계 개요 (하이브리드 3단)

```
위기지수 = 100 − 수급동향지표(교사신호/모델예측)     ← 높을수록 위기
   (A) 분위수 기반 기본단계  →  (B) 규칙 오버라이드(격상)  →  (C) 히스테리시스(진동방지)
```

- **(A) 분위수**: 광종별 위기지수 히스토리 분위수로 기본단계 부여 (광종 차등)
- **(B) 규칙 오버라이드**: 과업 요구(특정 변수 단독 트리거)를 반영해 격상
- **(C) 히스테리시스**: 하향은 2주 연속 낮을 때만 반영 → 경보 깜빡임(flapping) 방지

## 2. 자원안보특별법 붙임2 정성기준 → 정량규칙 매핑

| 단계 | 고시 정성기준(요지) | 정량 규칙(구현) |
|---|---|---|
| **관심** | 생산국 정세불안, 재난징후, 가격변동성 증가 | 위기지수 ≥ P50, 또는 변동성 ≥ P90 |
| **주의** | 정세불안 증가, 수급위기 가능성, 변동성 급증 | 위기지수 ≥ P70, 또는 지정학 severity ≥ 0.85 발생 |
| **경계** | 전쟁발발·시설파괴, 수송로 일부봉쇄, 조달 일부차질 | 위기지수 ≥ P85, 또는 **수입편중 HHI ≥ P90**(단독 트리거) |
| **심각** | 정세악화, 대형재난, 전면봉쇄, 조달 전면차질 | 위기지수 ≥ P95 (복합 극단) |

> 오버라이드 예시(과업 명시): "희토류는 특정국 수입편중도가 일정 수준 넘으면 즉시 '경계' 격상" → `import_hhi ≥ P90 → 최소 경계`. 지정학 severe 이벤트(수출통제·제재·전쟁) 발생 주는 **1단계 격상(최소 주의)**.

## 3. 광종별 임계값 (위기지수 분위수, 자동 산출)
분위수 컷은 광종별 히스토리에서 자동 계산(정상<P50, 관심 P50, 주의 P70, 경계 P85, 심각 P95). 광종마다 분포가 달라 절대 컷이 다르게 적용됨(예: CU는 지표 변동폭이 커 심각 컷이 높음).

## 6. 한계 / 다음 단계
- **분위수는 상대기준**: 현재 컷은 광종별 히스토리 상대분위수. 운영 시 공단이 **자원안보특별법 절대기준**(예: 조달차질률·비축일수)으로 캘리브레이션 필요.
- **운영 입력**: 지금은 실제 수급동향지표를 위기지수로 사용. 운영에선 **진단모델 예측값**을 입력(모델은 이미 위기분류 AUC 0.977).
- **지정학 트리거 정밀화**: LLM 전면추출로 severity·이벤트유형 품질 향상 시 오버라이드 정확도↑.
- **검증**: 공단 제시 시나리오·과거 실제 경보 이력과 대조(민감도·안정성).
