## B. 「필수데이터_부족분_정리」 객관성 검증

기존 문서의 각 주장을 원본으로 대조:

| 기존 문서 주장 | 검증 결과 | 판정 |
|---|---|---|
| 🔴 월간 수입통계 없음(관세청 연간뿐) | 관세청·UN Comtrade **모두 연간** 확인 | ✅ 정확 |
| 🔴 **세계 소비량 자체가 없음(USGS는 생산·매장만)** | **EU SCRREEN factsheet에 소비량 존재**: 구리 "global apparent consumption 25Mt(2021)", 코발트 "world refined cobalt consumption ~175,000t(2021)" 등 | ❌ **과기술(오류)** |
| 🟠 **희토류(REE) 가격 미보유** | KOMIS 가격시트에 **네오디뮴·란탄·디스프로슘·터븀·세륨 월간가격 존재**, 공급망통계에 산화물 6종. **DB에도 이미 적재됨**(fact_indicator PRICE=REE, 각 154행) | ❌ **과기술(오류)** |
| 🟠 한국 GDP·산업생산 없음(중국만) | fact_series 15종에 한국 거시 없음, 원본에도 부재 | ✅ 정확 |
| 🟡 생산·매장 장기 시계열 없음(USGS 2024~25만) | fact_production_reserve: 생산 2024~25, 매장 2025만 | ✅ 정확 |
| 🟡 배터리·EV 수요지표 없음 | 원본 파일 검색 결과 부재 | ✅ 정확 |

### 정정 필요 사항 (2건)
1. **세계 소비량**: "없음" → **"정형 시계열은 없으나 EU SCRREEN factsheet(광종별 PDF)에 글로벌/EU 소비량 수치 존재. 특정연도·비정형이라 시계열화 필요."** → 변수④(공급/소비)는 *부분 가용*으로 재분류.
2. **희토류 가격**: "미보유" → **"주간 LME 가격은 없으나 KOMIS 월간 원소가격(네오디뮴 등) 보유·DB적재됨. REE는 월간 단위 모델링 가능."** → REE 제외는 데이터 부재가 아니라 *주간패널 설계상 제외*였음.

> **결론(B):** 6개 주장 중 **4개 정확, 2개(세계소비량·희토류가격) 과기술**. 두 항목은 실제로 부분/전부 보유하고 있어 필수 부족분에서 **하향 조정** 필요.

## C. 추가 확인 사실 (참고)
- `EU SCRREEN 광종별 보고서`(60 PDF): 5대 중 **구리·니켈·리튬·코발트 factsheet 존재**, **희토류 그룹 factsheet는 없음**(스칸듐만). → REE 소비량은 SCRREEN에도 제한적.
- `핵심광물 수급위기 진단체계` 폴더 = 전체 원본 zip **중복본**(신규 데이터 아님).
- `sample` 폴더 = 원본 일부 **샘플**(신규 데이터 아님).
