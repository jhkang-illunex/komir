# 진단모델 베이스라인 학습 리포트

> 인수인계 컨텍스트(A: 지정학 지표 모델가용화 → B: 진단모델 학습) 중 **B 완료**.
> 데이터: `mart_weekly_diagnosis` 2020년~ → **월간 패널 312행**(CO·CU·LI·NI, REE 제외).
> 타깃: KOMIS 수급동향지표(교사신호). **낮을수록 위기.** 검증: 시간순 홀드아웃(학습 ~2024 / 검증 2025~).

## 0. 선행 작업 A — 지정학 지표 모델가용화 (완료)
인수인계 문서의 3대 갭을 해소:
- **광종 연결**: 규칙 텍스트매칭 + 문서힌트로 백필 → 태깅 이벤트 126 → **913건**(NI 511·CU 238·LI 135·CO 16·REE 13). 나머지는 거시성(중동전쟁·미중관세) → 별도 `geo_macro` 팩터로 활용.
- **시간 커버리지**: KOMIS 월간 116건 추가 분석(pymupdf 고속추출) → 지정학 이벤트 **2019~2026 전구간** 확보(2020~24 공백 해소). geo_event 1,742 → **6,357건**.
- **mart 재조인**: 월간 `fact_geopolitical_monthly`(광종별) + `fact_geopolitical_macro_monthly`(거시) 생성 → ASOF로 주간 mart에 결합. **변수⑥ 커버리지 1.9% → 학습구간 100%**.

## 1. 모델 설정
- **피처(9)**: 시장변동성①, 수입HHI②, 수입YoY·CAGR3③, 생산HHI⑤, Cash-3M스프레드, **지정학⑥·거시geo**, 기준가 + 광종(범주)
- **분할**: train 240행(2020~2024) / test 72행(2025~). 무작위 CV 금지(월→주 ASOF 자기상관 회피 위해 월간 집계 + 시간순).
- **프레이밍 2종**: (a) 회귀(지표값 예측) (b) 위기 이진분류(지표<20)

## 2. 결과

### (a) 회귀 (검증 2025~)
| 모델 | MAE | RMSE | R² |
|---|---|---|---|
| Naive(광종 평균) | 17.81 | 19.79 | 0.278 |
| Ridge(+광종더미) | 16.43 | 19.04 | 0.331 |
| **HistGBM(+광종)** | **14.28** | 19.43 | 0.303 |

→ GBM이 MAE 최저(나이브 대비 −20%). R²는 0.3대(월간 지표가 완만·평활되어 절대값 회귀는 난도 높음).

### (b) 위기 이진분류 (지표<20) — **핵심**
- **AUC = 0.977** (검증 2025~, 위기비율 0.21)
- 즉, "지금이 위기인가"를 **매우 잘 구분**. 과업의 4단계 경보 산출과 직결되는 실사용 프레이밍.

## 3. 해석 (중요)
- **수입편중도(HHI)·가격·스프레드가 지배적.** 교사신호(KOMIS 수급동향지표)가 본질적으로 **가격 기반**이라, 가격·교역 피처가 이를 잘 재현함(EDA와 일치).
- **지정학⑥ 중요도가 낮은 건 예상된 결과.** 지정학은 KOMIS 지표를 *재현*하는 데 기여가 작을 뿐, KOMIS가 *놓치는 급성 이벤트*(DRC 코발트 수출중단 등) 포착이 본래 역할. → 지정학은 "회귀 정확도"가 아니라 "경보 조기·보강"으로 평가해야 함.
- **광종마다 관계 부호가 뒤집히는 문제**(EDA 발견)를 GBM(광종 포함)이 흡수 → 선형 대비 개선.
