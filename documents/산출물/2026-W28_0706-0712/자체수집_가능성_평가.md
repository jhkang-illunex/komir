# 요청 데이터 항목 — 자체 수집 가능성 평가

> 대상: `데이터_요청서_내부검토표`의 부족 데이터. 각 항목을 **자체(공개 데이터/API) 수집 가능성**으로 분류.
> 원칙: 핵심 소스는 **웹으로 실제 확인**(할루시네이션 배제). 확인 못 한 항목은 "확인 필요"로 표기.
> 범례: 🟢가능(공개) · 🟡부분(일부 공개·가공 필요) · 🔴불가(발주처/내부/상용전용)

---

## 1. 🟢 자체 수집 가능 (공개 데이터/API — 확인 완료)

| 항목 | 공개 소스 | 방법 | 유의사항 |
|---|---|---|---|
| **B-1 월간 관세청 수출입통계** + **A-7 월간 국가별 수입** | **관세청 "품목별 국가별 수출입실적" OpenAPI**(공공데이터포털) | HS 2/4/6/10단위 × 국가 × **월별** 조회. `apis.data.go.kr/1220000/nitemtrade/...` | 무료 인증키. 매월 15일경 전월 현행화. **예측모델 월간 타깃을 자체 확보 가능** |
| **B-2 한국 GDP·산업생산** | **한국은행 ECOS OpenAPI** | GDP·산업생산지수 등 시계열 조회 | 무료 인증키, **출처표시 시 상업적 사용 무료**. (보조: 통계청 KOSIS) |
| **A-6 세계 생산·매장 장기 시계열** | **USGS** Historical Statistics(1900~, ~90광종) · Mineral Commodity Summaries(1996~) · Data Series 896 | 다운로드/파싱(보유 폴더에 USGS 2019~2026 PDF도 있음) | 공개. 생산·매장·교역·**소비·재고**까지 포함(美 중심) |
| **A-1 지정학 원천** + **A-3 가격급등 이벤트 이력** | 공개 뉴스·정책·공시 + **보유 가격데이터** | ① 보유 가격에서 급등시점 자동 도출 ② 뉴스 스크래핑(자체 지정학 파이프라인 이미 구축) | 상용 보고서(Argus 등) 외 공개소스로 상당부분 대체 가능 |

## 2. 🟡 부분 수집 가능 (일부 공개·가공/확인 필요)

| 항목 | 가능 부분 | 한계 |
|---|---|---|
| **A-2 세계 소비량** | USGS Historical Statistics에 **소비·재고 포함**(美+일부 글로벌) · 산업협회(구리 ICSG·니켈 INSG·코발트 Cobalt Institute) · IEA · EU SCRREEN(보유) | **글로벌 정형 시계열**은 협회별 산재·일부 유료. 통합 시계열화 작업 필요 |
| **A-8 ESG/인권 지수** | RMI(책임광물)·Amnesty·공개 ESG 자료로 **구성** 가능 | 바로 쓸 "지수" 미존재 → 자체 산정식 설계 필요 |
| **B-4 배터리·EV 수요지표** | EV: IEA Global EV Outlook·국내 EV 등록(국토부/KAMA) 공개 | 배터리 셀 상세(SNE Research 등)는 상용 유료 가능성 → 확인 필요 |

## 3. 🔴 자체 수집 불가 (발주처/내부/상용전용)

| 항목 | 사유 |
|---|---|
| **0-1 희토류 대상 "1종" 특정** | 과업 **범위(대상) 결정** 사항 → 발주처 확정 필요(수집 불가) |
| **A-5 과거 자원안보위기 경보 발령 이력** | 산업부/공단 **내부 공식 기록** → 발주처만 제공 가능 |
| **B-3 예측모델 필수 포함 변수 확정** | 발주처 **지정** 사항(의사결정) |
| **A-4 희토류(REE) 주간가격** | 신뢰 주간가격은 대부분 **상용 유료**(Asian Metal·Argus·SMM 등). 월간은 이미 보유 |

---

## 4. 시사점

- **핵심 병목 2건이 사실상 자체 확보 가능**: 예측모델 타깃인 **월간 수입통계(B-1)** 는 관세청 OpenAPI로, **한국 GDP·산업생산(B-2)** 은 한국은행 ECOS로 즉시 수집 가능 → **발주처 제공을 기다리지 않고 예측모델 월간화 착수 가능.**
- **생산·매장 장기시계열(A-6)** 도 USGS 공개데이터로 확장 가능(변수⑤ 시계열 보강).
- **지정학·이벤트(A-1·A-3)** 는 보유 데이터 + 자체 스크래핑으로 상당부분 대체.
- **발주처 필수는 4건으로 축소**: 희토류 1종 결정, 경보 이력, 예측 필수변수, (REE 주간가격은 상용구매 여부 결정).

> 권고: 요청서에서 🟢·🟡 항목은 **"자체 수집 예정(공개 API)"**로 표기하고, 발주처에는 🔴 4건 중심으로 요청 → 요청 범위를 줄이고 착수 속도를 높임. 단, **관세청/ECOS API 사용 시 인증키 발급·이용약관(출처표시)** 확인 필요.
